**A quick, easy way to add posts to page**

== Description ==

= You can display posts as =
* Unordered list 

Also work with Gutenberg shortcode block.

**Note:** After inserting shortcode or widget to site, if you are not seeing any post list then kindly visit blog/post section on frontend. It will store the post count on the database** 

<code>[recent_posts]</code>

**Limit:**
[recent_posts posts="5"] (Display no of post in a list)

= Here is Template code =
<code> <?php echo do_shortcode('[recent_posts]'); ?> </code>
